# 🌟 Task 1 Completed – Celebrating Telugu Culture Website (ApexPlanet Internship)

A journey that began with code, ended with culture 🌱

## 🛕 Project Title:
**"Celebrating Telugu Culture" – A Digital Homage to Heritage**

## ✨ Objective:
To design a culturally immersive, aesthetically rich website using HTML, CSS, and JavaScript that reflects the essence of Telugu traditions, cuisine, festivals, and values — bringing culture to the web.

---

## 🎯 Project Highlights:

### 🏛️ Cultural Sections:
- **Traditions & Rituals**: A deep dive into age-old customs and practices.
- **Cuisine**: Displaying flavorful and authentic Telugu dishes.
- **Festivals**: Vibrant celebrations with visual richness.
- **Media Playlists**: Curated cultural songs, videos, and devotional tracks.
- **Tools Used**: Transparent tech showcase for learners and devs.
- **Interactive Quotes**: Button-triggered Telugu quotes with bell sounds.

---

## 💻 Tech Stack Used:
- **HTML5**: Semantic layout and structure
- **CSS3**: Traditional color palette (deep red, saffron, gold), textured backgrounds, elegant fonts, gradients, and subtle animations
- **JavaScript**: DOM manipulation, temple bell interaction, Telugu quote logic

---

## 🎨 Design Elements:
- Smooth scroll and transitions
- Fully responsive layout
- Temple bell sound effect
- Quote pop-ups with spiritual touch
- Culturally inspired styling with modern design aesthetics
